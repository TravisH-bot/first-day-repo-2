Notes

  • ID's are unique and should be used to style individual, one-off elements.
  • Classes should be used to style multiple objects the same way.
  • Each line represents a declaration.
  • In a declaration, you have a property and a value.
  • You can add multiple classes to an HTML element but they must have a space between each one.
